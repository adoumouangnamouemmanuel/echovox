# Speech Support Research Website

## Quick Start

```bash
# Install dependencies
npm install

# Run development server
npm run dev
```

Visit http://localhost:3000

## Pages Created

✅ Home - About the Project
✅ Problem Space
✅ Research Approach
✅ Current Findings & Insights
✅ Ethics & Privacy
✅ Project Timeline (with visual timeline component)
✅ Team
✅ Contact & Collaboration

## Key Features

- Professional research documentation design
- Full navigation with mobile responsiveness
- Ethical disclaimers throughout
- Timeline visualization component
- Contact form for inquiries
- Accessibility-focused design
- SEO-optimized metadata

## Technology

- Next.js 16 (App Router)
- TypeScript
- Tailwind CSS
- Custom UI components
- Lucide React icons
- Framer Motion (for subtle animations)

## Important Notes

This website is a research documentation platform, not a medical tool or therapeutic solution. It clearly communicates:

- The project is ongoing research
- Not a treatment or finalized product
- Conducted with ethical oversight
- Privacy and anonymity protected
- Not enrolling participants
